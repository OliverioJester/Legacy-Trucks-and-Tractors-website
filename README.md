# Legacy Trucks and Tractors Website

A commercial website developed for **Legacy Trucks and Tractors Inc.**, a sister company of **Powertrac Inc.** The website serves as the company's official online presence, providing visitors with information about the company's products, services, and contact details through a clean and responsive interface.

## Tech Stack

- Laravel
- Tailwind CSS
- Alpine.js

## Project Overview

This website was designed and developed within **one month** using Laravel, Tailwind CSS, and Alpine.js. The project focuses on delivering a fast, secure, and user-friendly experience while maintaining a clean, minimalist design that reflects the company's professional branding.

## Highlights

- ⚡ Optimized for fast loading and performance
- 🔍 SEO-friendly structure and metadata
- 🛡️ Built with security best practices in mind
- 📱 Fully responsive across desktop, tablet, and mobile devices
- 🎨 Clean and minimalist user interface
- ♿ Accessibility-conscious design
- 🧩 Modular and maintainable Laravel architecture

The website was created as a commercial project to establish the online presence of **Legacy Trucks and Tractors Inc.**, enabling customers to easily explore the company's products and services.
